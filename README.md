# ESP32-C3 Multi-Sensor IoT Hub & Data Logger
hardware system based on ESP32-C3 (RISC-V architecture) on a 4-layer PCB for noise immunity. 

The project includes sensors (environment, light, and sound), smart power and charging management (MCP73871), and a user interface.
The system supports local data storage via SD card and external flash memory via SPI connection, with Wi-Fi and BLE connectivity.


1) All-in-One Sensing Solution: Integrated temperature, humidity, light, and sound sensors, eliminating the need for multiple external modules.
2) Industrial-Grade Reliability: The 4-layer design ensures the device operates without interference in noisy industrial environments.
3) Seamless Power Integration: Built-in USB-C charging and battery management, meaning your product is portable and user-friendly from day one.
4) Rapid Manufacturing: Delivered with a full production package (Gerbers, BOM, Pick-and-Place), optimized to minimize assembly errors and part costs.
