# ESP32-IoT-Sensor-Hub-Manufacturing-Ready-
hardware system based on ESP32-C3 (RISC-V architecture) on a 4-layer PCB for noise immunity. The project includes sensors (environment, light, and sound), smart power and charging management (MCP73871), and a user interface. system supports local data storage via SD card and external flash memory via SPI connection, with Wi-Fi and BLE connectivity.
